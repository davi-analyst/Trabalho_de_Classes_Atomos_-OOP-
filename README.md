# Câmbio de Átomos -OOP-

## Conceitos Aplicados
O projeto foca em pilares fundamentais do Python:

Encapsulamento: Uso de atributos privados (convenção _atributo).

Decoradores @property: Implementação de Getters e Setters para controle de acesso aos dados.

Composição de Objetos: A classe Atomo é composta por instâncias das classes Proton, Neutron e Eletron.

Modelagem de Dados: Representação física de propriedades como carga elétrica e massa atômica.

## Como funciona
O código modela as três partículas fundamentais e as agrupa em uma estrutura de átomo. Através dos métodos setter, é possível atualizar dinamicamente as propriedades do elemento (como transformar Hidrogênio em Hélio ou Lítio) e refletir essas mudanças no estado do objeto.
